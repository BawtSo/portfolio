What are Semantic Elements?

Semantic elements are tags of code that help developers format a webpage.

What are the different types off semantic elements?

Header

The header is the introductory content for the information being provided

Footer

Footer provides the information about the page creation, relative information about the owner, and policies the content is covered by

Section

An element which contains information related to the content of the page

Article

An element that contains information that can be independently used outside the scope of the page

Nav

Nav is an element on a page that provides navigation links

What is a webpage?

Semantic elements are an expanded definition of elements used in HTML5, which is the most recent markup language constructed for creating html documents. Semantic elements are tags of code that help the user and developer identify the type of information being used when creating an html document. These elements are used in a format that help build and structure a webpage. The benefit of using semantic elements is the concise definition each tag provides for the information being passed.
